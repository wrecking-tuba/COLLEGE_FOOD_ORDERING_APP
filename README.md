# College Food Ordering App — Delicatessen

Delicatessen is an Android-based college food ordering application designed to help students and faculty pre-order food from college canteens or food stalls.

The goal of the project is to reduce waiting time at physical counters, improve order management, and provide a smoother food ordering experience within a college campus.

## Project Overview

Traditional canteen ordering requires students to stand in queues, place orders manually, wait for preparation, and then collect the food. This application reduces that waiting time by allowing users to order food directly from their Android devices.

The app also provides a chef/admin interface to manage dishes, update menus, and track customer orders.

## Key Features

- Student/customer registration and login
- Chef/admin login
- View available food menu
- Add food items to cart
- Place food orders
- Track order status
- Chef can post new dishes
- Chef can update or delete menu items
- Customer profile management
- Secure login using user credentials

## User Roles

### Customer / Student

Customers can:

- Register and log in
- Browse available dishes
- Add dishes to cart
- Place orders
- Track order status
- View and update profile information

### Chef / Admin

Chef/admin users can:

- Log in securely
- Add new dishes to the menu
- Update existing dishes
- Delete dishes
- View and manage customer orders
